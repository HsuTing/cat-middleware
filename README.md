# Cat-middleware
Middleware of koa server.

## Middleware
- koa-react: render `html` with `react` and `nunjucks`.
- koa-relay: render `html` with `react`, `relay` and `nunjucks`.
- See more information in [wiki](https://github.com/HsuTing/cat-middleware/wiki).
